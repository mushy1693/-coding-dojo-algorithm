Assignment: Replace_String

Create a function called replace_string that does exactly what the replace function does in Javascript.
Do this without using split function but only using if/else, for loops, basic array manipulation with pop, push, length/count.

Example

Return a string where "Microsoft" is replaced with "CodingDojo":

var str = "Visit Microsoft!";
var res = replace_string("Microsoft","CodingDojo", str);

The result of res will be:

Visit CodingDojo!
